## project overview

I had been kicking around the idea of making my own PIC programmer for a bit after I started looking for a new job. I thought it would be good to get a better understanding of what's happening on my favorite microcontroller when the IDE uploads the program. Then, after a call with the hiring manager for a position I applied for that I really wanted, I found out I would be presenting on a project the following week. So I pulled the trigger, hoping I could get something working in a few days. I ended up being too optimistic, but eventually was able to get things working.

I'll try to give some details on the programming protocol this PIC uses, how I chose to actually implement the thing, and also talk about some problems that came up during development.

[Git repo](https://github.com/bsmitty5000/pic_programmer).
